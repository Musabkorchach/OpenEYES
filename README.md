# OpenEyes

> **See • Understand • Remember • Act**

**OpenEyes** is an open-source AI vision platform for robots and intelligent systems.

It helps machines see the world, understand objects and scenes, remember what they have seen, and act through robotics-ready APIs.

---

## Mission

To build an open-source AI vision platform that enables robots and intelligent systems to perceive, understand, remember, and interact with the real world.

---

## Vision

To become one of the most accessible open AI vision platforms for robotics, automation, education, smart devices, and community-driven innovation.

---

## What OpenEyes Can Do Today

- Real-time camera vision
- Face detection
- Eye detection
- YOLO object detection
- Object memory engine
- Multi-language Pi app interface
- GitHub-based open-source development

## Core Philosophy

OpenEyes follows four principles:

### See

Detect people, objects, faces, eyes, hands, motion, and visual patterns.

### Understand

Analyze scenes, relationships, behaviors, and context.

### Remember

Build short-term, working, and long-term visual memory.

### Act

Send decisions and commands to robots, drones, smart devices, and applications.
